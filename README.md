Sunbeam Connect — Intelligent Volunteer Ecosystem
Sunbeam Connect — интеллектуальная платформа для управления волонтерской деятельностью в Казахстане. Система автоматизирует подбор кадров, прогнозирует риски и внедряет прозрачную систему репутации.

Основные функции
AI Matchmaking: Семантический подбор задач на основе навыков волонтера с использованием вексельных эмбеддингов (Gemini text-embedding-004).

Attendance Predictor: Прогноз вероятности явки волонтеров на основе истории активности и сложности мероприятия.

Voice-to-Task: Создание задач через голосовой ввод с автоматическим парсингом данных (дата, место, требуемые навыки) в JSON.

Reputation Engine: Система уровней (1–50) и XP, рассчитываемая на основе отзывов кураторов и подтвержденных фото-отчетов.

Analytics Dashboard: Тепловая карта активности волонтеров и прогноз дефицита навыков для администраторов фонда.

Технологический стек
Frontend: React, TypeScript, Tailwind CSS, Framer Motion.

Backend: Supabase (Auth, PostgreSQL, pgvector).

AI: Google Gemini 1.5 Flash (Vision, Processing, Embeddings).

Localization: i18next (Kazakh, Russian, English).
